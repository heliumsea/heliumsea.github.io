Preprocessing

210x160 bitmap with a 128-color =>
encode a single frame we take the maximum value for each pixel color value over the frame being encoded and the previous frame, because some objects only occur in either even frames or odd frames. =>
extract Y channel (black-white image, see https://en.wikipedia.org/wiki/YUV) =>
rescale it to 84x84

Model acchitecture

Input: 84x84x4
1st Conv Layer: 32 filters of 8x8 with stride 4 with ReLu
2nd Conv Layer: 64 filters of 4x4 with stride 2 with ReLu
3nd Conv Layer: 64 filters of 3x3 with stride 1 with ReLu
4nd FC Layer: 512 units with ReLu
Output: FC, action-size
