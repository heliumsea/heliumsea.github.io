Reinforcement Learning 01: Basics

2021-04-27 RL

Overview

Reinforcement Learning is used to solve problems in this framework: Agents interact with the environment as well as other agents, and we would like the agents to achieve some objectives.
For example, to control a robot to collect all coins in a maze.
We call the process that an agent completes the objective an episode.
An episode consists of many steps, and in each step, the agent gets the state from the environment, takes action, and gets a reward value.
Our objective is to get a policy telling us when the agent at a state, how possibly it takes different actions.
